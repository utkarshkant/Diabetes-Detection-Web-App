# Diabetes Detection Web App

## Objective
- Build a machine learning model to detect diabetes
- Make a web app from that model

## Data
Source: https://www.kaggle.com/saurabh00007/diabetescsv

### Features
- Pregnancies
- Glucose    
- BloodPressure             
- SkinThickness             
- Insulin                   
- BMI                       
- DiabetesPedigreeFunction  
- Age                       
- Outcome
